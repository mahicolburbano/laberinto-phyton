# laberinto-phyton
# Laberinto de Texto en Python

Este proyecto consiste en un juego de laberinto de texto creado en Python. El jugador navegará a través de un laberinto representado en ASCII y deberá encontrar la salida mientras evita chocar con las paredes. ¡Diviértete explorando el laberinto!

### Funcionalidades:
- Movimiento del jugador a través del laberinto.
- Representación del laberinto en ASCII.
- Detección de colisiones con las paredes del laberinto.

### Instrucciones de Uso:
1. Ejecuta el archivo main.py.
2. Sigue las instrucciones en pantalla para mover al jugador.
3. Encuentra la salida del laberinto.

### Requisitos:
- Python 3.x

¡Disfruta del juego!
